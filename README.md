# Weather Forecast Application 

## Description
A responsive Weather Forecast web application built using HTML, Tailwind CSS, and JavaScript.
It fetches real-time weather data using the OpenWeatherMap API.


## Features
- Search weather by:
  - City name
  - Pincode
  - Village / Mandal / District / State / Country
- Current location weather using GPS
- Interactive map-based weather selection
- Temperature unit toggle (°C / °F)
- Dynamic background based on weather conditions
- Extreme temperature alerts
- Recently searched locations dropdown
- 5-day extended weather forecast
- Responsive UI (Desktop, iPad Mini, iPhone SE)


## Technologies Used
- HTML5
- Tailwind CSS
- Vanilla JavaScript
- OpenWeatherMap API
- Leaflet.js (Map integration)


## How to Run the Project
1. Clone or download the repository
2. Open `index.html` in any modern browser
3. Add your OpenWeatherMap API key inside `app.js`
4. Search for a location or use the map



## API Used
OpenWeatherMap  
https://openweathermap.org/api
